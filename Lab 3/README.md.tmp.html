Fibonacci and Towers of Hanoi

Two algorithms for calculating the first n numbers of Fibonacci sequence, one is iterative and the other is rcursive.

A recursive algorithm for calculating the moves for n disks to solve Towers of Hanoi.

  • Fibonacci.java
  • Hanoi.java
  • Performance graphs.
  • Recursion practical sheet.

javac Fibonacci.java

java Fibonacci
